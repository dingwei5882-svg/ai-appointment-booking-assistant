# AI Appointment Booking Assistant

A modern landing page for an AI-powered appointment booking assistant, built with Astro and Tailwind CSS.

## Features

- **Hero Section**: A welcoming hero section to introduce the service.
- **Feature List**: Highlights key features of the service.
- **Workflow Steps**: A clear, step-by-step explanation of how appointment booking automation works.
- **Appointment Form**: A booking audit request form for visitors to request a free appointment automation audit.
- **Client-Side Validation**: Basic form validation to ensure required fields are filled.

## Tech Stack

- **Astro**: The web framework for building fast, content-driven websites.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **TypeScript**: For type safety and better developer experience.
- **Node.js**: The runtime environment.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm

### Running Locally

1. Clone the repository.
2. Navigate to the project directory:
   ```bash
   cd ai-appointment-booking-assistant
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:4321`.

### Building for Production

To create a production build, run the following command:

```bash
npm run build
```

This will generate the static files in the `dist/` directory.
